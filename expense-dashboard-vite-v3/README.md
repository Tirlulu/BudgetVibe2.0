
# דשבורד הוצאות אשראי – Vite + React (גרסה 3)

## מה כולל
- העלאת דוחות אשראי (`.xlsx`) + פרסינג אוטומטי (איתור כותרת "תאריך רכישה" וכו').
- טעינת קטגוריות/Aliases מקובץ `קטגוריות_מאוחדות_טיוטה.xlsx` (או גרסה ערוכה שלך).
- סיווג אוטומטי היברידי: חוקים של משתמש → Aliases → Keywords.
- מסך "דורש סיווג" ליצירת חוקים (נשמרים מקומית – LocalStorage).
- פילטר לפי כרטיס (XXXX), KPI's, מגמות חודשיות, התפלגות לפי קטגוריות, חיובים חוזרים.

## הרצה
```
npm install
npm run dev
```
פתח את http://localhost:5173 והעלה את קבצי האשראי + קובץ הקטגוריות.

## הערות
- כללי המשתמש נשמרים בדפדפן (LocalStorage). בפתרון שרת – החלף לשמירה ב־DB.
- הפרסר מניח דוחות עם כותרות בעברית בסגנון שהצגת. אם יש הבדלי פורמט – נעדכן את מאתר הכותרת/עמודות.
